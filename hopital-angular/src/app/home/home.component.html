Tous les codes Angular de notre application se trouvent dans le répertoire src/app.
Tous les composants créés se trouvent dans src/app/nom-composant.
La classe Visite se trouve dans src/app/visite.ts.
Tous les noms de fichiers sont en kebab-case, alors que le nom des classes sont en CamelCase.
